## 1. 启动前置条件

- `CP_CHAT_AUTH_JWT_SECRET` 必须提供，且长度至少 32 位
- MySQL / Redis / MinIO 必须可连接
- 本地 Docker 依赖建议先通过 `docker-up` 启动并等待健康

## 2. 推荐流程

1. 先启动外部依赖
2. 再启动应用
3. 若需要后台运行，使用后台启动脚本
